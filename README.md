# SolarLab PV Emulator Pro

A one-page professional educational PV simulator for Iraq/Kurdistan training labs.

## Run locally

```bash
npm install
npm run dev
```

## Cloudflare Pages

- Build command: `npm install && npm run build`
- Build output directory: `dist`
- Root directory: `/`
- Environment variable: `NODE_VERSION=20`

Do not upload `node_modules`, `dist`, or `package-lock.json` to GitHub.

## Safety

This simulator is for education and virtual training only. It is not certified engineering design software and must not be used as the only basis for real electrical installation.
